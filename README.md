## Setup Project

- Node.js 18, 20 or 22. 
- Require: npm, chrome
- step 1 

```bash
$ npm init playwright@latest
$ yarn run "test:shopee"

```
- step 2

```bash
$ npx playwright install
$ yarn run "test:shopee"

```

## Execute test

Execute command `npm test:<project name>`

Example run test shopee:

```bash
$ npm run "test:shopee"
$ yarn run "test:shopee"

```
